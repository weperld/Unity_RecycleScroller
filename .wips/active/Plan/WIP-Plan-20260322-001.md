## 수정 설계

### A. RecycleScroller.cs — 캐시 필드 추가

`#region Data Cache` 영역에 필드 추가:

```csharp
// UpdateCellView 분기 캐싱 (LoadData 시 설정)
private Vector2 m_cachedAxisVec;          // Vector2.up or Vector2.right
private Vector2 m_cachedWidthMaskVec;     // (1,0) for vertical, (0,1) for horizontal
private Vector2 m_cachedContentPosVec;    // Vector2.up for vertical, Vector2.left for horizontal
private RectTransform m_cachedTopSpaceCell;
private RectTransform m_cachedBottomSpaceCell;
```

### B. RecycleScroller_LoadData.cs — 캐시 초기화

`UpdateObjectsOnLoadData()` 내에서 `UpdateCellView()` 호출 전에 캐시 설정 메서드 호출:

```csharp
private void CacheUpdateCellViewValues()
{
    m_cachedAxisVec = ScrollAxis == eScrollAxis.VERTICAL ? Vector2.up : Vector2.right;
    m_cachedWidthMaskVec = ScrollAxis == eScrollAxis.VERTICAL ? new Vector2(1f, 0f) : new Vector2(0f, 1f);
    m_cachedContentPosVec = ScrollAxis == eScrollAxis.VERTICAL ? Vector2.up : Vector2.left;
    m_cachedTopSpaceCell = Rt_TopSpaceCell;
    m_cachedBottomSpaceCell = Rt_BottomSpaceCell;
}
```

호출 시점: `Init → InitializeOnLoadData → CalculateScrollValues → UpdateObjectsOnLoadData(CacheUpdateCellViewValues → UpdateCellView)`

### C. UpdateCellView 수정

#### C-1. ScrollAxis 분기 제거

| 기존 | 수정 |
|------|------|
| `ScrollAxis == eScrollAxis.VERTICAL ? contentPos.y : -contentPos.x` | `Vector2.Dot(contentPos, m_cachedContentPosVec)` |
| `ScrollAxis == eScrollAxis.VERTICAL ? Vector2.up : Vector2.right` | `m_cachedAxisVec` |
| `switch (ScrollAxis) { ... }` widthVec 설정 | `Vector2.Scale(Viewport.rect.size, m_cachedWidthMaskVec)` |

#### C-2. Rt_TopSpaceCell/Rt_BottomSpaceCell 프로퍼티 → 캐시 필드

UpdateCellView 내에서만 `m_cachedTopSpaceCell`/`m_cachedBottomSpaceCell` 사용.
기존 public 프로퍼티는 외부 호환을 위해 유지.

#### C-3. m_reverse 분기 → 진입부 1회 계산

UpdateCellView 진입부에서 로컬 변수로 1회 정리:

```csharp
// m_reverse 기반 값 1회 계산
```

findVisibleGroupIndices 이후:


#### C-4. LINQ 내 m_reverse 조건 단순화
